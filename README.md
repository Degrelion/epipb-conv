# 🌐 Groupe Discord - E.P.I.P.B

Bienvenue dans la conversation de l'E.P.I.P.B ! Ce projet vise à stalker (de façon fictive) les membres de l'.E.P.I.P.B afin de trouver des indices sur un certain « M ».

*Toute ressemblance avec un organisme ou des individus réels serait purement fortuite.**

## ✨ Fonctionnalités

- **Design Responsive**: Fonctionne parfaitement sur ordinateur.
- **Interface**: Interface propre et intuitive, similaire à Discord.
- **Animations**: Animations fluides et agréables pour les différentes interactions.
- **Navigation Channel**:  Barre latérale pour accéder aux différents canaux.
- **Monopage**: Certains contenu est disponible avec fonctionnalité de messagerie.

## 🛠 Technologies Utilisées

- **HTML5**: Pour la structuration de la page web.
- **CSS3**: Pour les styles et animations.
- **JavaScript**: Pour l'interactivité et comportement dynamique.
